# CMYS Fight 天赋与 Buff 系统设计案

## 一、系统定位

新的 Buff 系统不应只回答"角色伤害提高了多少"，而应持续改变玩家的决策：

* 使用哪把武器。
* 何时换枪。
* 何时闪避。
* 是否主动摧毁建筑。
* 是否进入隐藏房。
* 选择哪个章节出口。
* 是否接受污染与异常。
* 如何形成跨系统协同。

系统分为四层：

1. **记忆天赋**：远征前在主城选择，一局只能携带一个。
2. **局内协议 Buff**：远征过程中获得，构成主要 Build。
3. **协议进化与命名协同**：满足组合条件后自动激活或在设施中完成进化。
4. **异常 Buff**：强力但具有明确代价，主要来自花园、隐藏房和污染路线。

---

# 二、基础规则

## 2.1 Buff 容量

玩家最多持有：

* 12 个普通协议 Buff。
* 1 个记忆天赋。
* 2 个异常 Buff。
* 1 个协议进化。

异常 Buff 不占普通 Buff 槽位，但会提高污染值。

## 2.2 稀有度

| 稀有度 | 定位                         |
| --- | -------------------------- |
| 普通  | 单一、清晰地改变一个动作或属性            |
| 优良  | 将两个动作或系统连接起来               |
| 稀有  | 明显改变战斗方式或地图决策              |
| 进化  | 由同系列 Buff 数量触发，不直接从普通池随机获得 |
| 异常  | 强效果与强代价并存                  |

不建议再让"传奇"只是更高数值的普通 Buff。传奇级内容统一改为**协议进化**。

## 2.3 不重复获得

普通 Buff 不可重复获得。

获得已经拥有的 Buff 时，应转换为以下选择之一：

* 精炼该 Buff。
* 获得重投次数。
* 转换为金币或废料。
* 从同系列另外两个 Buff 中选择。

## 2.4 构筑保护

为了避免连续拿到无关 Buff：

* 玩家尚未形成第一个协同时，当前已持有标签的权重约提高至 1.8 倍。
* 形成第一个协同后降低至约 1.35 倍。
* 形成第二个协同后恢复正常。
* 连续两次选择未出现任何相关标签时，下一次至少包含一个相关选项。
* 每组三选一至少包含一个"改变机制"的 Buff，不能全部是纯数值。

## 2.5 Buff 来源

| 来源      | 主要产出             |
| ------- | ---------------- |
| 普通战斗进度  | 普通、优良            |
| 精英房     | 优良、稀有            |
| Boss    | 稀有、进化机会          |
| 无伤 Boss | 稀有选择、进化核心、最大资源   |
| 商店      | 可预见的普通与优良 Buff   |
| 隐藏房     | 协同组件、地图 Buff、异常  |
| 挑战裂隙    | 高品质 Buff 或精炼     |
| 许愿泉     | 有代价的定向选择         |
| 照相亭     | 闪避、换枪、动作触发类 Buff |
| 广播终端    | 控制后续房间或奖励倾向      |
| 工坊      | 精炼、拆解、协议进化       |

# 三、记忆天赋

记忆天赋在重生泉选择，负责决定本局初始倾向，而不是直接给予永久高额属性。

## 1. 弹道记忆

第一次 Buff 选择必定包含一个先锋协议。

获得第一把非起始武器后，该武器的投射物速度提高 10%，持续至本章结束。

## 2. 回响记忆

初始获得一次档案脉冲。

第一次完美闪避后，立刻恢复 25%闪避冷却。

## 3. 凤凰记忆

每章第一次护甲完全破裂时，清除角色附近的小型敌方投射物。

若没有护甲，则在生命首次降至一半以下时触发。

## 4. 星图记忆

每次章节路线选择时，提前完整显示其中一个出口的：

* 敌人倾向。
* 环境危险。
* 奖励倾向。
* 威胁等级。

另一个出口只显示部分信息。

## 5. 武库记忆

在军械回廊选择一个武器家族。

本局第一次武器奖励显著倾向该家族，但不保证出现指定武器。

## 6. 拾荒记忆

每章前五个被摧毁的场景物额外增加废料计量。

第一次完成废料计量时，必定掉落能量与少量金币。

## 7. 秘径记忆

每章第一次进入存在隐藏房的普通房间时，播放一次微弱方向提示。

不会直接显示入口位置。

## 8. 禁忌记忆

初始污染值增加 15。

本局第一个隐藏房或异常祭坛的奖励品质提高一级。

# 四、普通协议 Buff

普通 Buff 分为六个系列：

* 先锋协议：武器与投射物。
* 回响协议：闪避、移动和换枪。
* 凤凰协议：生命、护甲和防御转化。
* 以太协议：技能与能量循环。
* 拾荒协议：场景建筑、资源和设施。
* 星图协议：地图、隐藏房和路线出口。

---

# 五、先锋协议

## 1. 弹道校准器

**稀有度：普通**

* 散布降低 25%。
* 暴击率提高 5%。

保留现有 `scattershot` 的定位，但名称与实际效果更加一致。

## 2. 扳机纪律

**稀有度：普通**

持续射击超过 0.8 秒后：

* 投射物速度提高 15%。
* 散布逐渐降低。
* 停止射击后效果在 0.5 秒内消退。

替代单纯永久提高射速的设计。

## 3. 相位弹药

**稀有度：优良**

* 投射物穿透一个敌人。
* 每次穿透后伤害降低 10%。

高单发武器收益高，但不会无限贯穿。

## 4. 折返弹道

**稀有度：优良**

* 投射物可以从墙面反弹一次。
* 反弹后的投射物伤害提高 20%。

房间建筑布局会直接影响该 Buff 的强度。

## 5. 余烬载荷

**稀有度：优良**

玩家投射物会施加燃烧。

* 燃烧持续 2.4 秒。
* 重复命中刷新持续时间，但不无限叠层。
* 可点燃孢子、油污和部分场景设施。

## 6. 处决窗口

**稀有度：稀有**

生命低于 25%的敌人会被标记。

* 对标记敌人伤害提高 20%。
* 击杀标记敌人恢复 1 点能量。
* Boss 只在阶段生命低于 15%时生效。

---

# 六、回响协议

该系列依赖计划中的通用战术闪避，但不同角色可以拥有不同闪避距离、无敌时间和附加效果。

## 1. 反相反击

**稀有度：普通**

完美闪避后，下一次攻击：

* 伤害提高 35%。
* 投射物速度提高 20%。

效果持续三秒。

## 2. 残像摄影

**稀有度：优良**

完美闪避后，在原位置留下持续 1.5 秒的残像。

* 残像会吸引部分敌方投射物。
* 同一时间只能存在一个。
* 内置冷却六秒。

适合与照相亭建立直接联系。

## 3. 交叉换装

**稀有度：普通**

切换武器后：

* 新武器获得 1.2 秒攻击速度提升。
* 返还少量能量消耗。
* 内置冷却三秒。

鼓励双武器使用，而不是始终使用单一最强武器。

## 4. 交替电流

**稀有度：优良**

两把不同武器在两秒内交替命中敌人时，获得一层同步。

获得四层后：

* 向附近敌人释放一次连锁电弧。
* 同步层数清空。

## 5. 动量储蓄

**稀有度：优良**

连续移动 1.5 秒后，下一次攻击获得：

* 一次额外穿透。
* 更高击退。
* 命中后产生小范围冲击。

停止移动或受击会失去蓄力。

## 6. 擦弹继电器

**稀有度：稀有**

近距离避开敌方投射物会积累擦弹计数。

每五次擦弹：

* 恢复八点能量。
* 缩短当前闪避冷却 0.5 秒。

同一枚投射物只能计算一次。

---

# 七、凤凰协议

## 1. 强化心核

**稀有度：普通**

* 最大生命提高二点。
* 立即恢复二点生命。

对应现有 `reinforced_heart`。

## 2. 护甲电容

**稀有度：普通**

* 最大护甲提高二点。
* 立即恢复二点护甲。

对应现有 `armor_capacitor`。

## 3. 快速复充

**稀有度：普通**

护甲恢复延迟降低 35%。

若角色本身没有护甲恢复，则获得缓慢的基础护甲恢复能力。

## 4. 反应式装甲

**稀有度：优良**

护甲完全破裂后：

* 获得两秒的投射物伤害减免。
* 对附近敌人造成一次小型击退。
* 每个房间最多触发一次。

## 5. 应急屏障

**稀有度：稀有**

每章一次，受到致命伤害时：

* 保留一点生命。
* 立即摧毁角色附近的小型投射物。
* 三秒内不能再次获得无敌类保护。

比单纯"免死一次"更具视觉和操作反馈。

## 6. 壁垒记忆

**稀有度：优良**

靠近实体建筑、掩体或大型结构时：

* 护甲恢复速度提高 50%。
* 受到的击退降低。
* 离开建筑附近后效果立即结束。

让新加入的场景建筑真正影响 Build。

---

# 八、以太协议

## 1. 电容单元

**稀有度：普通**

* 最大能量提高八点。
* 立即恢复八点能量。

对应现有 `capacitor_cell`。

## 2. 流量调节器

**稀有度：普通**

能量自然恢复速度提高 25%。

对应现有 `flux_regulator`。

## 3. 技能加速器

**稀有度：优良**

技能冷却时间降低 20%。

对应现有 `skill_accelerator`。

## 4. 能量反馈

**稀有度：普通**

使用技能后，延迟 0.5 秒恢复五点能量。

同一技能的多段效果不会重复触发。

## 5. 熵引擎

**稀有度：优良**

击杀敌人恢复两点能量。

每秒最多恢复八点，避免小型敌群造成无限循环。

## 6. 连锁指令

**稀有度：稀有**

使用技能后，下一次攻击会额外连接附近一个敌人，造成原伤害的 35%。

* 不会再次触发连锁。
* 三秒内未攻击则失效。

---

# 九、拾荒协议

## 1. 战场拾荒

**稀有度：普通**

每摧毁五个可破坏场景物，生成一个补给球。

补给球在能量和金币之间交替。

每个房间最多触发两次。

## 2. 拆迁分红

**稀有度：优良**

敌人被以下效果击杀时，额外掉落少量资源：

* 环境陷阱。
* 爆炸桶。
* 熔炉。
* 孢子爆炸。
* 倒塌建筑。

Boss 和召唤单位不会反复掉落。

## 3. 临时工事

**稀有度：优良**

摧毁大型可破坏物时，会留下持续 2.5 秒的临时掩体。

* 阻挡敌方投射物。
* 玩家和敌人可以绕过。
* 同时最多存在两个。

## 4. 危险解析镜

**稀有度：普通**

* 玩家受到的环境伤害降低 30%。
* 敌人受到的环境伤害提高 20%。

不影响敌我普通攻击。

## 5. 终端越权

**稀有度：稀有**

广播终端、遗产装置和地牢工坊：

* 增加一个可选项目。
* 每章获得一次设施重投。
* 部分高风险选项会提前显示污染代价。

## 6. 回收协议

**稀有度：优良**

未拾取的武器可以被拆解为：

* 金币。
* 废料计量。
* 少量能量。

稀有武器拆解时额外获得一个 Buff 重投碎片。

三个碎片合成一次完整重投。

---

# 十、星图协议

## 1. 秘径感知

**稀有度：普通**

进入存在隐藏连接的房间时，环境音会出现轻微变化。

清理房间后，入口所在大致方向短暂闪烁。

不会直接标记具体墙块。

## 2. 回声制图

**稀有度：普通**

每新探索三个房间，随机揭示一个相邻未知房间的奖励类别。

只显示：

* 武器。
* Buff。
* 商店。
* 恢复。
* 异常。
* 精英。

## 3. 路线评估

**稀有度：优良**

检查章节出口时，额外显示一项信息：

* 主要敌人标签。
* 环境标签。
* 奖励标签。
* 威胁变化。

如果已经全部显示，则提前显示该路线的特殊房倾向。

## 4. 绕路收益

**稀有度：普通**

每个关卡第一次清理不位于最短出口路径上的战斗房时，获得：

* 十二金币，或
* 四点能量。

每关只能触发一次。

## 5. 档案钥匙

**稀有度：优良**

每章第一次遇到以下门时可以免费开启：

* 金币门。
* 普通封印门。
* 资源门。

不能免费开启：

* 无伤门。
* 高难度门。
* 剧情能力门。

## 6. 完美勘测

**稀有度：稀有**

无伤清理战斗房后：

1. 优先将附近隐藏连接从"隐藏"提升为"传闻"；
2. 若附近没有秘密，则揭示一个未知房间的奖励；
3. 若地图信息已经完全公开，则获得一个重投碎片。

# 十一、协议进化

同一系列拥有三个 Buff 后，会在以下时机触发进化机会：

* Boss 奖励。
* 工坊。
* 特殊进化祭坛。
* 高级隐藏房。

每局只能完成一次协议进化。

## 1. 超频核心

**条件：三个先锋协议**

进入超频状态的条件：

* 连续命中六次，或
* 在短时间内击杀三名敌人。

超频持续三秒：

* 暴击率提高 15%。
* 散布降低 30%。
* 投射物速度提高 20%。
* 首次触发时恢复少量能量。

## 2. 幽灵回路

**条件：三个回响协议**

完美闪避后：

* 残像复制玩家接下来的两次攻击，造成 35%伤害。
* 切换武器返还 30%当前闪避冷却。
* 该冷却返还三秒内只能触发一次。

## 3. 神盾铸造厂

**条件：三个凤凰协议**

* 最大护甲提高四点。
* 护甲恢复过程中持续产生微弱排斥力。
* 护甲完全恢复时获得一秒弹幕屏障。
* 屏障每八秒最多产生一次。

## 4. 技能闭环

**条件：三个以太协议**

每第三次使用技能时：

* 0.6 秒后重复一次弱化版本。
* 重复技能造成 40%效果。
* 不会再次触发能量反馈或其他"使用技能"事件。

## 5. 活体地牢协议

**条件：三个拾荒协议**

每个战斗房的第一个活性场景装置可以被过载：

* 孢子囊扩大爆炸范围。
* 冷却终端延长冻结。
* 熔炉提高伤害。
* 档案设施释放电弧。

环境击杀还会加速废料计量。

## 6. 星界图册

**条件：三个星图协议**

* 所有相邻房间显示奖励类别。
* 路线出口完整显示威胁等级。
* 每章第一个隐藏出口至少会显示为"传闻"。
* 不直接显示隐藏入口的位置。

# 十二、命名协同

命名协同由特定两个 Buff 组合触发，不占槽位。

## 1. 灼热折射

**折返弹道 + 余烬载荷**

* 反弹后的投射物会留下短暂燃烧轨迹。
* 每枚投射物第一次反弹时生成三枚弱化火星。

## 2. 镜面穿刺

**相位弹药 + 折返弹道**

投射物若先穿透敌人再撞墙，或者先反弹再命中敌人：

* 伤害提高 25%。
* 保留剩余穿透和反弹次数。

## 3. 先手处决

**扳机纪律 + 处决窗口**

对被标记敌人的第一轮连续射击：

* 更快进入稳定射击状态。
* 第一发必定暴击。
* 对 Boss 每阶段最多触发一次。

## 4. 残像齐射

**残像摄影 + 反相反击**

完美闪避产生的残像会复制玩家的反击攻击。

复制攻击造成 40%伤害，但不触发新的残像。

## 5. 交叉电流

**交叉换装 + 交替电流**

每次换枪立即获得两层同步。

连锁电弧触发后，两把武器各恢复少量能量。

## 6. 危险驾驶

**动量储蓄 + 危险解析镜**

* 穿越环境危险不会中断动量。
* 蓄力攻击命中活性场景物时会立即触发其效果。

## 7. 壁垒回路

**壁垒记忆 + 快速复充**

靠近建筑完成护甲恢复时，向面对方向清除一小片敌方投射物。

每六秒最多触发一次。

## 8. 凤凰余烬

**应急屏障 + 余烬载荷**

应急屏障触发时：

* 点燃附近敌人。
* 恢复两点护甲。
* 在原位置留下短暂余烬区域。

## 9. 闭环施法

**能量反馈 + 连锁指令**

连锁攻击每命中一个额外目标，恢复两点能量。

单次最多恢复六点。

## 10. 熵回收

**熵引擎 + 战场拾荒**

击杀敌人有一定概率计为一次场景物摧毁。

环境击杀额外恢复两点能量。

## 11. 秘密买路

**秘径感知 + 档案钥匙**

* 秘密墙所需破坏量降低。
* 第一个隐藏房中的收费设施价格降低。
* 首次发现隐藏出口时获得少量金币。

## 12. 完美档案

**完美勘测 + 绕路收益**

无伤清理绕路战斗房时：

* 同时获得绕路奖励。
* 获得一个重投碎片。
* 若该房间连接隐藏区域，立即显示模糊入口提示。

# 十三、异常 Buff

异常 Buff 来自：

* 静思花园誓约。
* 地牢许愿泉。
* 污染商店。
* 隐藏祭坛。
* 特殊章节路线。
* 诅咒门。

异常 Buff 必须让玩家明确知道代价。

## 1. 血契引擎

* 最大生命降低二点。
* 所有直接伤害提高 20%。
* 无伤清理房间时恢复一点生命。

最大生命不能因此低于三点。

## 2. 堵塞信号

* 立即增加二十污染。
* 精英敌人出现率提高。
* 精英奖励品质提高。
* 精英敌人弹速略微增加。

## 3. 玻璃档案

* 伤害提高 35%。
* 玩家受到的直接攻击伤害提高一点。
* 环境伤害不受该额外伤害影响。

## 4. 空壳装甲

* 最大护甲提高四点。
* 护甲无法在战斗中自然恢复。
* 护甲完全破裂时释放一次大范围清弹。
* 清理房间后恢复一点护甲。

## 5. 诅咒罗盘

* 普通出口不再显示奖励倾向。
* 隐藏房和隐藏出口出现率提高。
* 隐藏区域奖励品质提高一级。
* 观星台只能恢复部分路线信息。

## 6. 熔炉心脏

* 站在环境危险附近时快速恢复能量。
* 环境伤害会绕过护甲，直接作用于生命。
* 环境击杀提高下一次攻击伤害。

## 7. 黑色账本

商店允许欠债购买。

每产生一笔债务：

* 增加污染。
* 下一位 Boss 的最大生命提高。
* 偿还全部债务后获得一次商店折扣。

## 8. 回声饥饿

每进入一个此前未访问的房间：

* 伤害提高 3%。
* 移动速度提高 1%。

最多叠加五层。

返回已访问房间或使用快速传送会清空层数。

# 十五、隐藏房与 Buff 的关系

隐藏房可以细分为：

## 普通密室

偏向提供：

* 金币。
* 能量。
* 重投碎片。
* 普通或优良 Buff。

## 协同密室

根据玩家当前标签，提供能够接近协同的选项。

例如持有折返弹道时，提高以下内容的出现率：

* 余烬载荷。
* 相位弹药。
* 动量储蓄。
* 场景反弹类武器。

## 异常祭坛

提供三种有代价选择：

* 失去最大生命换取稀有 Buff。
* 增加污染换取协议进化。
* 移除一个普通 Buff，换取两个同系列选择。

## 挑战裂隙

奖励在入口处提前展示。

接受奖励后必须完成：

* 弹幕挑战。
* 环境陷阱。
* 限制闪避次数。
* 无伤击杀。
* 连续换枪挑战。

完成后返回原房间。

# 十六、出口房与天赋互动

出口房不只是结束关卡，也应成为构筑检查点。

普通出口显示：

* 章节名称。
* 敌人标签。
* 环境标签。
* 奖励倾向。
* 威胁等级。

特定 Buff 可以改变出口：

* 路线评估：增加一项信息。
* 星界图册：完整显示所有普通出口。
* 诅咒罗盘：隐藏普通路线信息，但提高秘密路线概率。
* 完美勘测：无伤表现可能揭示隐藏出口。
* 档案钥匙：免费开启部分资源型路线门。
* 黑色账本：允许欠费进入收费出口，但增加污染。

隐藏出口不能只要求固定 Buff。

应当同时提供替代条件，例如：

```text
相位透镜
或
支付三十金币
或
污染达到四十
或
完成本关三次无伤战斗
```

这样构筑会影响进入方式，但不会形成绝对死门。

# 十九、推荐的数据结构

```ts
type BuffFamily =
  | "vanguard"
  | "echo"
  | "phoenix"
  | "aether"
  | "salvage"
  | "survey";

type BuffTrigger =
  | "onShoot"
  | "onHit"
  | "onKill"
  | "onDodge"
  | "onPerfectDodge"
  | "onWeaponSwap"
  | "onSkillUsed"
  | "onArmorBreak"
  | "onPlayerHurt"
  | "onPropDestroyed"
  | "onHazardKill"
  | "onRoomEntered"
  | "onRoomPerfectClear"
  | "onBossPerfectClear"
  | "onFacilityUsed"
  | "onExitInspected";

interface BuffDefinition {
  id: string;
  name: string;
  rarity: "common" | "uncommon" | "rare";
  family: BuffFamily;
  tags: string[];
  triggers?: BuffTrigger[];
  synergyKeys?: string[];
  routeBias?: string[];
  description: string;
}
```

不要继续把所有逻辑堆积在：

```ts
if (BuffSystem.has(player, "..."))
```

建议建立事件分发层，再由 Buff 系统返回对应效果。

# 二十一、最终设计原则

1. 普通 Buff 改变一个动作。
2. 优良 Buff 连接两个动作。
3. 稀有 Buff 改变一个循环。
4. 协同 Buff 改变组合规则。
5. 协议进化改变整套 Build。
6. 异常 Buff 改变风险结构。
7. 地图 Buff 只能帮助发现秘密，不能成为发现秘密的唯一方法。
8. 路线 Buff 只能影响信息和代价，不能直接指定最优路线。
9. 环境 Buff 必须利用实际场景建筑，而不是增加隐藏百分比。
10. 每个 Buff 都应产生可以被玩家直接感知的视觉或行为变化。
11. 纯数值 Buff 应保持少量，并主要承担构筑基础。
12. 高品质 Buff 不应只是基础 Buff 的数值放大版本。

这套系统最终形成的核心体验是：

> 玩家不是在收集十二个增伤百分比，而是在逐步建立一套能够操纵武器、闪避、建筑、资源、地图与章节路线的战斗协议。
