# Michele Integration Reference

## Source identity

- Character page: https://wiki.biligame.com/klbq/米雪儿·李
- Cross-reference page: https://strinova.org/wiki/Michele
- Role cues retained: guardian, attacker reveal, deployable cat turret, automatic rifle.

## Game adaptation

The source game uses a team-shooter ruleset, while cmys-fight is a 320 × 240 single-player roguelite. The implementation therefore preserves the character fantasy rather than copying competitive timings directly.

- Pawtector: deploys an eight-second automated turret with an 88-pixel acquisition range. It fires every 0.35 seconds and applies a short slow.
- Cat Trace: damage from an enemy marks that attacker for two seconds and gives it auto-aim priority.
- Inspector: Michele-exclusive automatic rifle. It gains 35% damage against the currently marked attacker.
- Michele is unlocked for existing and new profiles together with Inspector so the added content is immediately usable.

## Visual references

Temporary source images were used only to study Michele's blonde hair, blue/white investigator outfit, cat-ear equipment and Inspector's white/navy/cyan receiver. The repository contains newly authored low-resolution sprites and palettes, not downloaded source images.
